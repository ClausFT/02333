HOW TO RUN AND COMPILE THE CODE

1. In Eclipse, select File -> Import... -> Existing Projects into Workspace
2. Check the radiobutton "Select archive file" and select the file "s121666Databar1.zip"
3. Make sure "s121666Databar1" is checked in the checkbox
4. Click finish
5. Click the build button or alternatively hit CTRL-B to build the project
6. Click the run button or alternatively hit CTRL-F11 to run the project
